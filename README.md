# Core Java Programs Collection

This repository contains beginner-friendly Core Java example programs demonstrating important Java concepts such as:

- Methods
- Constructors
- Loops
- Interfaces
- Static Members
- Break & Continue Statements
- Recursion
- Method Overloading
- Exception Handling
